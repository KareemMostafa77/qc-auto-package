📝 Use in Tests (v2.0 Route-Based IDs)

🛣️ Route-based format: qc_{{ '{{' }}route{{ '}}' }}_{{ '{{' }}tag{{ '}}' }}_{{ '{{' }}identifier{{ '}}' }}
Example: On /home route → qc_home_button_saveBtn

Cypress

// Full ID (route-specific)
cy.get('[data-qcauto="qc_home_button_saveBtn"]').click();

// Pattern matching (route-independent)
cy.get('[data-qcauto$="button_saveBtn"]').click();

// All buttons on home route
cy.get('[data-qcauto^="qc_home_button"]').should('exist');

Playwright

// Full ID
await page.locator('[data-qcauto="qc_home_form_demoForm"]').fill('test');

// Route-independent
await page.locator('[data-qcauto*="form_demoForm"]').fill('test');
