🎯 What it does

ng-qcauto automatically adds data-qcauto attributes to your DOM elements. Testers can configure which elements to track via localStorage - no code changes needed!

🧪 Try it now (3 simple steps)

Step 1: Configure what to track

Press F12, open Console, paste this command and press Enter:

localStorage.setItem('qcAuto-tags', JSON.stringify(['button', 'input', 'form']));

Step 2: Reload the page

Type this in console or press Ctrl+R:

location.reload();

Step 3: Inspect elements below

Right-click on any button/input below → "Inspect Element" → Look for data-qcauto attributes!
